# Project Setup

# Description

<h3> Image Processing API using : </h3>

      - Node.Js
      - sharp
      - Jasmine
      - prettier
      - eslint

# Running

<h2> Install dependencies </h2>

     npm install

<h2> Run development server </h2>

     npm run start

<h2> Testing  </h2>

     npm run test

<h2> Compile typescript to javascript </h2>

     npm run build

<h2> Linting </h2>

     npm run lint
     npm run lint:fix

<h2> Formatting code with Prettier </h2>

     npm run format
